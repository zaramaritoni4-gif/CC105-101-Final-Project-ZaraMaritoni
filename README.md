# Resort Booking System

## Description
A simple yet elegant Resort Booking System built with **HTML, CSS, JavaScript, and MySQL**.  
Manages **rooms, guests, and bookings** with interactive forms and tables.  
Designed to be visually appealing, user-friendly, and fully functional for basic booking operations.

## Technologies Used
- **HTML**: Semantic structure for pages  
- **CSS**: Responsive, colorful, and modern layout  
- **JavaScript**: Client-side validation and interactive features  
- **MySQL**: Backend database with relational tables
- 
1. Open the HTML files in a browser.  
2. Use the forms to add new entries as needed.  
3. Tables display some example data.  
4. Optionally, run the SQL script in MySQL to see all data.

- **Challenges:** Creating a readable and visually distinct system, validating date inputs correctly, and managing relational data integrity and testing
